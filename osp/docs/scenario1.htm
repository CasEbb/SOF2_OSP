    Scenario : Setting up a competitive match

    1. Get on an OSP server fair to both clans.


    2. If you're an admin, make sure the server is setup with competitive settings, you can callvote the default settings by doing /adm comp if you aren't an admin, but are an admin you can do /ref comp (if novote_comp is not set or set to 0, i.e.: comp voting is enabled), callvote yourself to admin if it is enabled, if all else fails, you can /callvote g_dowarmup 2 if it is enabled to at least be in ready-up mode.


    3. If you are an admin and not part of a neutral party (i.e.: league official, friend of the two clans playing, etc), you MUST do an /adm suspend, other players can make sure that no non neutral players aren't admins by doing the /players command, it will show the status of everyone on the server.


    4. If you're a referee you can set the two team's settings using the referee commands, or each team can set their own settings using the team commands.


    5. Everybody readies up (/ready), or someone calls a start_match vote(/callvote start_match), or one member of each teams readies his whole team (/tcmd ready), or the referee readies everyone (/ref ready) and the countdown starts.


    6. The match ensues, if the scorelimit or timelimit is set appropriately and you want to swap teams when it either is reached, if match_swap is enabled the teams will be auto-swapped, if match_fastRestart is enabled, the map will quickly be restarted and will not cycle to the next map in the map rotation.
